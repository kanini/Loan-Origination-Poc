# User Story: Dashboard Page with Recent Extractions and Statistics

**Epic:** EP-003 - Dashboard & Home Page  
**Story ID:** US-003-001  
**Priority:** P0 - Critical  
**Estimate:** 8 Story Points (3-4 days)

---

## User Story

**As a** Financial Analyst  
**I want** a dashboard landing page with recent document activity, quick statistics, and upload CTA  
**So that** I can quickly access my recent work and start new document processing efficiently

---

## Acceptance Criteria

### AC1: Dashboard Layout Structure
**Given** I navigate to the Dashboard (/)  
**When** the page loads  
**Then**:
- Welcome message "Welcome to GenAI PDF Extractor" is displayed (H1)
- Prominent "Upload New Document" CTA button is centered at top
- Page uses 3-column grid layout on desktop (≥1024px)
- Page uses 2-column layout on tablet (768px-1023px)
- Page uses stacked single-column layout on mobile (<768px)
- Maximum content width is 1280px (centered)

### AC2: Upload Call-to-Action
**Given** the Dashboard is displayed  
**When** I view the hero section  
**Then**:
- Large primary button "Upload New Document" with upload icon
- Button size is Large (48px height)
- Button uses Primary Blue background (#3B82F6)
- Button has hover effect (scale 1.02)
- Clicking button navigates to Upload page (/upload)
- Button is keyboard accessible

### AC3: Recent Extractions Widget
**Given** I have processed documents  
**When** I view the Recent Extractions section  
**Then**:
- Section heading "Recent Extractions" (H2)
- Last 5-10 documents displayed as cards
- Each card shows: thumbnail, filename (truncated), upload date, status badge, model used
- Cards are displayed in grid (3 columns desktop, 2 columns tablet, 1 column mobile)
- Clicking a card navigates to Results page for that document
- "View All" link navigates to History page

### AC4: Document Card Details
**Given** a document card is displayed  
**When** I view the card content  
**Then**:
- PDF thumbnail or placeholder icon (64×64px)
- Filename truncated to 40 characters with ellipsis
- Upload date formatted as "MMM DD, YYYY" (e.g., "Jan 15, 2026")
- Status badge: Green for "Success", Red for "Failed", Blue for "Processing"
- Model used badge: "OpenAI GPT" or "Google Gemini"
- Card has shadow-sm and hover effect (shadow-md lift)

### AC5: Quick Statistics Section
**Given** the Dashboard is displayed  
**When** I view the Quick Statistics section  
**Then**:
- Section heading "Quick Statistics" (H3)
- Three stat cards displayed vertically or in grid
- Stat Card 1: "Total Documents Processed" with count number
- Stat Card 2: "Success Rate" with percentage (e.g., "95%")
- Stat Card 3: "Recent Activity" with count from last 7 days
- Each stat card has icon, label, and large value number

### AC6: Empty State
**Given** I have no processed documents  
**When** I view the Dashboard  
**Then**:
- Empty state illustration (document icon)
- Heading: "No documents processed yet"
- Description: "Upload your first PDF document to extract entity data and get started."
- Primary CTA: "Upload Your First Document" button
- Statistics show zeros (0 documents, 0% success rate, 0 recent activity)

### AC7: Loading State
**Given** the Dashboard is loading data  
**When** the page is fetching from API  
**Then**:
- Skeleton screens for document cards (5 skeletons)
- Skeleton screens for stat cards (3 skeletons)
- Skeletons match final card layout
- Header and navigation remain visible
- No blank screen during loading

### AC8: Error State
**Given** the API request fails  
**When** dashboard data cannot be loaded  
**Then**:
- Error icon displayed
- Message: "Failed to load dashboard. Please refresh the page or try again later."
- "Refresh" button to retry data fetch
- Statistics show fallback values (zeros or last cached data)

## Technical Details

### Component Structure
```typescript
// src/pages/Dashboard.tsx
import { useQuery } from '@tanstack/react-query'
import { Upload, FileText, TrendingUp, Activity } from 'lucide-react'
import { Button } from '@/components/ui/button'
import { Card } from '@/components/ui/card'
import { Badge } from '@/components/ui/badge'
import { Skeleton } from '@/components/ui/skeleton'
import { EmptyState } from '@/components/common/EmptyState'
import { usePageTitle } from '@/hooks/usePageTitle'
import { useAppNavigation } from '@/utils/navigation'
import { fetchRecentDocuments, fetchStatistics } from '@/api/documents'

export default function Dashboard() {
  usePageTitle('Dashboard')
  const { goToUpload, goToResults, goToHistory } = useAppNavigation()

  const { data: recentDocs, isLoading: docsLoading, error: docsError } = useQuery({
    queryKey: ['recent-documents'],
    queryFn: () => fetchRecentDocuments({ limit: 10 }),
  })

  const { data: stats, isLoading: statsLoading } = useQuery({
    queryKey: ['statistics'],
    queryFn: fetchStatistics,
  })

  if (docsLoading || statsLoading) {
    return <DashboardSkeleton />
  }

  if (docsError) {
    return <DashboardError onRetry={() => window.location.reload()} />
  }

  if (!recentDocs || recentDocs.length === 0) {
    return <DashboardEmpty onUpload={goToUpload} />
  }

  return (
    <div className="max-w-screen-2xl mx-auto px-4 md:px-6 lg:px-8 py-8">
      {/* Hero Section */}
      <div className="text-center mb-12">
        <h1 className="text-h1 font-bold text-gray-900 mb-4">
          Welcome to GenAI PDF Extractor
        </h1>
        <Button size="lg" onClick={goToUpload} className="gap-2">
          <Upload className="h-5 w-5" />
          Upload New Document
        </Button>
      </div>

      {/* Main Content Grid */}
      <div className="grid grid-cols-1 lg:grid-cols-3 gap-8">
        {/* Recent Extractions */}
        <div className="lg:col-span-2">
          <div className="flex items-center justify-between mb-4">
            <h2 className="text-h2 font-semibold text-gray-900">
              Recent Extractions
            </h2>
            <Button variant="tertiary" onClick={goToHistory}>
              View All
            </Button>
          </div>
          
          <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
            {recentDocs.map((doc) => (
              <DocumentCard
                key={doc.id}
                document={doc}
                onClick={() => goToResults(doc.id)}
              />
            ))}
          </div>
        </div>

        {/* Quick Statistics */}
        <div className="lg:col-span-1">
          <h3 className="text-h3 font-semibold text-gray-900 mb-4">
            Quick Statistics
          </h3>
          
          <div className="space-y-4">
            <StatCard
              icon={<FileText className="h-6 w-6 text-primary" />}
              label="Total Documents Processed"
              value={stats?.totalDocuments || 0}
            />
            <StatCard
              icon={<TrendingUp className="h-6 w-6 text-success" />}
              label="Success Rate"
              value={`${stats?.successRate || 0}%`}
            />
            <StatCard
              icon={<Activity className="h-6 w-6 text-secondary" />}
              label="Recent Activity (7 days)"
              value={stats?.recentActivity || 0}
            />
          </div>
        </div>
      </div>
    </div>
  )
}
```

### API Integration
```typescript
// src/api/documents.ts
import { apiClient } from './client'

export async function fetchRecentDocuments({ limit = 10 }) {
  const response = await apiClient.get('/api/documents/recent/', {
    params: { limit },
  })
  return response.data
}

export async function fetchStatistics() {
  const response = await apiClient.get('/api/documents/statistics/')
  return response.data
}
```
