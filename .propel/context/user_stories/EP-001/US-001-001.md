# User Story: React + TypeScript + Vite Project Setup

**Epic:** EP-001 - Frontend Foundation & Design System  
**Story ID:** US-001-001  
**Priority:** P0 - Critical  
**Estimate:** 5 Story Points

---

## User Story

**As a** Frontend Developer  
**I want** to set up a React 18+ project with TypeScript and Vite build tooling  
**So that** I have a modern, performant development environment with type safety for building the UI

---

## Acceptance Criteria

### AC1: Project Initialization
**Given** I need to create a new React frontend project  
**When** I initialize the project using Vite  
**Then** the project structure includes:
- React 18.3+ with TypeScript 5.3+
- Vite 5.0+ as build tool
- Standard folder structure: `src/components/`, `src/pages/`, `src/hooks/`, `src/utils/`, `src/types/`
- TypeScript configuration with strict mode enabled
- ESLint and Prettier configured

### AC2: Development Server
**Given** the project is initialized  
**When** I run `npm run dev`  
**Then** development server starts on `http://localhost:5173` with HMR

### AC3: API Proxy Configuration
**Given** the frontend needs to communicate with Django backend  
**When** development server runs  
**Then** `/api/*` requests proxy to `http://localhost:8000`
