# User Story: Drag-and-Drop File Upload with Validation

**Epic:** EP-004 - Document Upload Workflow  
**Story ID:** US-004-001  
**Priority:** P0 - Critical  
**Estimate:** 8 Story Points (3-4 days)

---

## User Story

**As a** Financial Analyst  
**I want** to upload PDF documents using drag-and-drop with file validation  
**So that** I can quickly select files for processing with immediate feedback on file validity

---

## Acceptance Criteria

### AC1: Drag-and-Drop Zone
**Given** I am on the Upload page  
**When** I view the file selection area  
**Then**:
- Large drag-and-drop zone with dashed border (400px × 300px minimum)
- Upload icon centered in zone
- Text: "Drag and drop your PDF file here, or click to browse"
- Subtext: "Maximum file size: 10MB"
- Zone has light gray background (#F9FAFB)
- Dashed border is Primary Blue (#3B82F6)

### AC2: Drag-Over Visual Feedback
**Given** I drag a file over the drop zone  
**When** the file is hovering over the zone  
**Then**:
- Border changes to solid Primary Blue
- Background changes to light blue (#EBF5FF)
- Border width increases to 2px
- Cursor shows "copy" indicator
- Visual feedback appears within 100ms

### AC3: File Drop Handling
**Given** I drop a file into the zone  
**When** the file is released  
**Then**:
- File is immediately validated for type and size
- If valid: File preview appears below zone
- If invalid: Error message displays with red styling
- Drop zone returns to default state
- Only one file can be selected at a time (previous file is replaced)

### AC4: Click-to-Browse Fallback
**Given** I prefer not to use drag-and-drop  
**When** I click anywhere in the drop zone  
**Then**:
- Native file browser dialog opens
- File filter shows only PDF files (*.pdf)
- After file selection, same validation occurs
- Selected file appears in preview

### AC5: File Type Validation
**Given** I select or drop a file  
**When** the file type is checked  
**Then**:
- Only PDF files (MIME type: application/pdf) are accepted
- Non-PDF files trigger error: "File type not supported. Please upload a PDF file"
- Error message has red text and error icon
- Drop zone shows red border
- File is not added to preview

### AC6: File Size Validation
**Given** I select or drop a PDF file  
**When** the file size is checked  
**Then**:
- Files ≤10MB are accepted
- Files >10MB trigger error: "File size exceeds 10MB limit"
- Error message shows file size (e.g., "File size: 12.5MB")
- Error has red text and error icon
- Drop zone shows red border
- File is not added to preview

### AC7: File Preview Display
**Given** a valid file is selected  
**When** validation passes  
**Then**:
- File preview card appears below drop zone
- Preview shows: PDF icon, filename, file size
- Filename is truncated if >50 characters with ellipsis
- File size formatted as "MB" or "KB" (e.g., "2.5 MB")
- Remove button (X icon) in top-right corner
- Preview has white background with shadow-sm

### AC8: Remove File Action
**Given** a file preview is displayed  
**When** I click the remove button (X)  
**Then**:
- File preview is removed
- Drop zone returns to default empty state
- File selection is cleared
- Model selection remains unchanged
- "Extract Entities" button becomes disabled

## Technical Details

### Component Implementation
```typescript
// src/components/upload/FileUploadZone.tsx
import { useCallback, useState } from 'react'
import { useDropzone } from 'react-dropzone'
import { Upload, X, FileText, AlertCircle } from 'lucide-react'
import { Button } from '@/components/ui/button'
import { cn } from '@/lib/utils'

interface FileUploadZoneProps {
  onFileSelect: (file: File | null) => void
  selectedFile: File | null
}

const MAX_FILE_SIZE = 10 * 1024 * 1024 // 10MB in bytes
const ACCEPTED_FILE_TYPE = 'application/pdf'

export function FileUploadZone({ onFileSelect, selectedFile }: FileUploadZoneProps) {
  const [error, setError] = useState<string | null>(null)

  const validateFile = (file: File): string | null => {
    if (file.type !== ACCEPTED_FILE_TYPE) {
      return 'File type not supported. Please upload a PDF file'
    }
    if (file.size > MAX_FILE_SIZE) {
      return `File size exceeds 10MB limit (File size: ${formatFileSize(file.size)})`
    }
    return null
  }

  const onDrop = useCallback((acceptedFiles: File[]) => {
    setError(null)
    
    if (acceptedFiles.length === 0) return
    
    const file = acceptedFiles[0]
    const validationError = validateFile(file)
    
    if (validationError) {
      setError(validationError)
      onFileSelect(null)
    } else {
      onFileSelect(file)
    }
  }, [onFileSelect])

  const { getRootProps, getInputProps, isDragActive } = useDropzone({
    onDrop,
    accept: { 'application/pdf': ['.pdf'] },
    maxFiles: 1,
    multiple: false,
  })

  const handleRemove = () => {
    setError(null)
    onFileSelect(null)
  }

  return (
    <div className="space-y-4">
      {/* Drop Zone */}
      <div
        {...getRootProps()}
        className={cn(
          "border-2 border-dashed rounded-lg p-12 text-center cursor-pointer transition-all",
          "hover:border-primary hover:bg-blue-50",
          isDragActive && "border-primary border-solid bg-blue-50",
          error && "border-error bg-red-50",
          selectedFile && "border-success bg-green-50"
        )}
      >
        <input {...getInputProps()} />
        
        <Upload className={cn(
          "h-12 w-12 mx-auto mb-4",
          isDragActive ? "text-primary" : "text-gray-400"
        )} />
        
        <p className="text-base text-gray-700 mb-2">
          Drag and drop your PDF file here, or click to browse
        </p>
        <p className="text-sm text-gray-500">
          Maximum file size: 10MB
        </p>
      </div>

      {/* Error Message */}
      {error && (
        <div className="flex items-start gap-2 p-4 bg-red-50 border border-error rounded-md">
          <AlertCircle className="h-5 w-5 text-error flex-shrink-0 mt-0.5" />
          <p className="text-sm text-error">{error}</p>
        </div>
      )}

      {/* File Preview */}
      {selectedFile && !error && (
        <div className="bg-white border border-gray-200 rounded-lg p-4 shadow-sm">
          <div className="flex items-start justify-between">
            <div className="flex items-start gap-3 flex-1">
              <FileText className="h-8 w-8 text-primary flex-shrink-0" />
              <div className="flex-1 min-w-0">
                <p className="text-base font-medium text-gray-900 truncate">
                  {selectedFile.name}
                </p>
                <p className="text-sm text-gray-500">
                  {formatFileSize(selectedFile.size)}
                </p>
              </div>
            </div>
            <Button
              variant="tertiary"
              size="sm"
              onClick={handleRemove}
              aria-label="Remove file"
            >
              <X className="h-4 w-4" />
            </Button>
          </div>
        </div>
      )}
    </div>
  )
}

function formatFileSize(bytes: number): string {
  if (bytes < 1024) return `${bytes} B`
  if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`
  return `${(bytes / (1024 * 1024)).toFixed(1)} MB`
}
```
