# User Story: LLM Model Selection Interface

**Epic:** EP-004 - Document Upload Workflow  
**Story ID:** US-004-002  
**Priority:** P0 - Critical  
**Estimate:** 3 Story Points (1 day)

---

## User Story

**As a** Financial Analyst  
**I want** to select between OpenAI GPT and Google Gemini models  
**So that** I can choose the most appropriate AI model for my document processing needs

---

## Acceptance Criteria

### AC1: Model Selection UI
**Given** I am on the Upload page with a file selected  
**When** I view the model selection section  
**Then**:
- Section heading "Choose LLM Model" (H3)
- Two model option cards displayed side-by-side on desktop
- Cards stacked vertically on mobile (<768px)
- Each card has radio button, model name, description, and icon
- Cards have white background with border and shadow-sm

### AC2: OpenAI GPT Model Card
**Given** the model selection is displayed  
**When** I view the OpenAI option  
**Then**:
- Radio button on the left
- Model name "OpenAI GPT" in bold (H4)
- Description: "Fast and accurate for standard documents"
- AI icon or OpenAI logo
- Card is clickable (entire card acts as radio button)
- Card has hover effect (border changes to Primary Blue)

### AC3: Google Gemini Model Card
**Given** the model selection is displayed  
**When** I view the Gemini option  
**Then**:
- Radio button on the left
- Model name "Google Gemini" in bold (H4)
- Description: "Advanced processing for complex layouts"
- AI icon or Gemini logo
- Card is clickable (entire card acts as radio button)
- Card has hover effect (border changes to Primary Blue)

### AC4: Model Selection Interaction
**Given** I interact with model cards  
**When** I click a model card  
**Then**:
- Radio button becomes selected
- Card border changes to Primary Blue (2px solid)
- Card background changes to light blue (#EBF5FF)
- Previously selected card is deselected
- Only one model can be selected at a time
- Selection state is maintained during page navigation

### AC5: Default State
**Given** I first arrive at the Upload page  
**When** no model is selected  
**Then**:
- Both cards have default gray border
- No radio button is checked
- "Extract Entities" button is disabled
- User must select a model before proceeding

### AC6: Keyboard Accessibility
**Given** I navigate using keyboard  
**When** I use Tab and arrow keys  
**Then**:
- Tab moves focus between model cards
- Arrow keys (↑↓) switch selection between models
- Space or Enter selects focused model
- Focus indicator is visible (2px outline)
- Screen reader announces model name and description

### AC7: Validation
**Given** I attempt to submit without selecting a model  
**When** I click "Extract Entities" button  
**Then**:
- Inline error message appears: "Please choose an LLM model"
- Model selection section has red border
- Focus moves to model selection area
- Button remains disabled

## Technical Details

### Component Implementation
```typescript
// src/components/upload/ModelSelection.tsx
import { useState } from 'react'
import { Sparkles } from 'lucide-react'
import { cn } from '@/lib/utils'

type ModelType = 'openai' | 'gemini' | null

interface ModelSelectionProps {
  selectedModel: ModelType
  onModelSelect: (model: ModelType) => void
  error?: string
}

const models = [
  {
    id: 'openai' as const,
    name: 'OpenAI GPT',
    description: 'Fast and accurate for standard documents',
    icon: Sparkles,
  },
  {
    id: 'gemini' as const,
    name: 'Google Gemini',
    description: 'Advanced processing for complex layouts',
    icon: Sparkles,
  },
]

export function ModelSelection({ selectedModel, onModelSelect, error }: ModelSelectionProps) {
  return (
    <div className="space-y-4">
      <h3 className="text-h3 font-semibold text-gray-900">
        Choose LLM Model
      </h3>

      <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
        {models.map((model) => {
          const Icon = model.icon
          const isSelected = selectedModel === model.id

          return (
            <button
              key={model.id}
              type="button"
              onClick={() => onModelSelect(model.id)}
              className={cn(
                "relative flex items-start gap-4 p-6 rounded-lg border-2 transition-all text-left",
                "hover:border-primary hover:bg-blue-50",
                "focus:outline-none focus:ring-2 focus:ring-primary focus:ring-offset-2",
                isSelected
                  ? "border-primary bg-blue-50"
                  : "border-gray-200 bg-white",
                error && !selectedModel && "border-error"
              )}
              role="radio"
              aria-checked={isSelected}
            >
              {/* Radio Button */}
              <div className="flex-shrink-0 mt-1">
                <div
                  className={cn(
                    "h-5 w-5 rounded-full border-2 flex items-center justify-center",
                    isSelected
                      ? "border-primary bg-primary"
                      : "border-gray-300 bg-white"
                  )}
                >
                  {isSelected && (
                    <div className="h-2 w-2 rounded-full bg-white" />
                  )}
                </div>
              </div>

              {/* Content */}
              <div className="flex-1">
                <div className="flex items-center gap-2 mb-2">
                  <Icon className="h-5 w-5 text-primary" />
                  <h4 className="text-h4 font-semibold text-gray-900">
                    {model.name}
                  </h4>
                </div>
                <p className="text-sm text-gray-600">
                  {model.description}
                </p>
              </div>
            </button>
          )
        })}
      </div>

      {/* Error Message */}
      {error && (
        <p className="text-sm text-error flex items-center gap-2">
          <AlertCircle className="h-4 w-4" />
          {error}
        </p>
      )}
    </div>
  )
}
```

### Usage in Upload Page
```typescript
// src/pages/Upload.tsx
import { useState } from 'react'
import { FileUploadZone } from '@/components/upload/FileUploadZone'
import { ModelSelection } from '@/components/upload/ModelSelection'

export default function Upload() {
  const [selectedFile, setSelectedFile] = useState<File | null>(null)
  const [selectedModel, setSelectedModel] = useState<'openai' | 'gemini' | null>(null)
  const [modelError, setModelError] = useState<string>('')

  const handleSubmit = () => {
    if (!selectedModel) {
      setModelError('Please choose an LLM model')
      return
    }
    // Proceed with upload
  }

  return (
    <div className="max-w-3xl mx-auto px-4 py-8">
      <h1 className="text-h1 font-bold mb-8">Upload Document</h1>
      
      <FileUploadZone
        selectedFile={selectedFile}
        onFileSelect={setSelectedFile}
      />
      
      <div className="mt-8">
        <ModelSelection
          selectedModel={selectedModel}
          onModelSelect={(model) => {
            setSelectedModel(model)
            setModelError('')
          }}
          error={modelError}
        />
      </div>
      
      <div className="mt-8 flex gap-4">
        <Button variant="secondary" onClick={() => navigate('/')}>
          Cancel
        </Button>
        <Button
          onClick={handleSubmit}
          disabled={!selectedFile || !selectedModel}
        >
          Extract Entities
        </Button>
      </div>
    </div>
  )
}
```
