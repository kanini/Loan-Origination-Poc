# User Story: Multi-Format Export (Excel, CSV, PDF)

**Epic:** EP-005 - AI Chatbot & Export Functionality  
**Story ID:** US-005-002  
**Priority:** P1 - High  
**Estimate:** 8 Story Points

---

## User Story

**As a** Loan Processor  
**I want** to export extracted entities in multiple formats (JSON, CSV, Excel, PDF)  
**So that** I can integrate the data into other systems and create reports

---

## Acceptance Criteria

### AC1: Export Dropdown
**Given** user is on Results page  
**When** they click Export button  
**Then** dropdown shows options:
- Download as JSON (implemented)
- Download as CSV
- Download as Excel
- Generate PDF Report

### AC2: JSON Export
**Given** user selects JSON export  
**When** export executes  
**Then** JSON file downloads with extraction data using Blob API

### AC3: CSV Export
**Given** user selects CSV export  
**When** export executes  
**Then** CSV file downloads with flattened entity data

### AC4: Excel Export
**Given** user selects Excel export  
**When** export executes  
**Then** Excel workbook downloads with:
- Client-side generation using SheetJS (xlsx library)
- Overview worksheet + separate worksheets per entity category
- Formatted headers: Bold, blue background (#3B82F6), white text, frozen pane
- Auto-sized columns to fit content
- Number formatting: Currency as `$#,##0.00`, Percentages as `0.00%`
- Filename pattern: `{document_name}_entities_{YYYY-MM-DD}.xlsx`

### AC5: PDF Report
**Given** user selects PDF report  
**When** export executes  
**Then** formatted PDF downloads with entity data and document reference

### AC6: Export Feedback
**Given** export is in progress  
**When** file is generating  
**Then**:
- Loading spinner on export button
- Success toast: "File downloaded successfully"
- Edge cases handled (empty categories = empty sheet with headers)
