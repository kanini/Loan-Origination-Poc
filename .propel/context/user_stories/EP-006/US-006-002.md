# User Story: PDF Viewer Integration with react-pdf

**Epic:** EP-006 - Results Display & Entity Visualization  
**Story ID:** US-006-002  
**Priority:** P0 - Critical  
**Estimate:** 5 Story Points (2 days)

---

## User Story

**As a** Financial Analyst  
**I want** to view the original PDF document alongside extracted entities  
**So that** I can verify extraction accuracy by comparing with the source document

---

## Acceptance Criteria

### AC1: PDF Viewer Component
**Given** I am on the Results page  
**When** the page loads  
**Then**:
- PDF viewer displays in right panel (40% width on desktop ≥1024px)
- PDF viewer in separate tab on tablet (768px-1023px)
- PDF viewer in collapsible section on mobile (<768px)
- PDF renders correctly with all pages
- Loading spinner shows while PDF loads
- Error message if PDF fails to load

### AC2: PDF Navigation Controls
**Given** the PDF is displayed  
**When** I interact with navigation controls  
**Then**:
- "Previous Page" button navigates to previous page
- "Next Page" button navigates to next page
- Page indicator shows "Page X of Y"
- First page: Previous button disabled
- Last page: Next button disabled
- Keyboard shortcuts: Arrow Left (previous), Arrow Right (next)

### AC3: Zoom Controls
**Given** the PDF is displayed  
**When** I use zoom controls  
**Then**:
- "Zoom In" button increases zoom by 25%
- "Zoom Out" button decreases zoom by 25%
- "Fit to Width" button fits PDF to container width
- "Fit to Page" button fits entire page in viewport
- Zoom level indicator shows percentage (e.g., "100%")
- Minimum zoom: 50%, Maximum zoom: 200%
- Keyboard shortcuts: Ctrl/Cmd + (zoom in), Ctrl/Cmd - (zoom out)

### AC4: Full-Screen Mode
**Given** the PDF is displayed  
**When** I click the full-screen button  
**Then**:
- PDF expands to full browser window
- Navigation and zoom controls remain visible
- Exit full-screen button appears
- Pressing Esc exits full-screen
- Full-screen works across all browsers

### AC5: PDF Loading State
**Given** the PDF is being loaded  
**When** I wait for the PDF to render  
**Then**:
- Loading spinner displayed in PDF panel
- Loading message: "Loading PDF..."
- Progress indicator if available
- Timeout after 30 seconds with error message
- Skeleton screen matches PDF viewer layout

### AC6: PDF Error Handling
**Given** the PDF fails to load  
**When** an error occurs  
**Then**:
- Error icon displayed
- Error message: "PDF preview unavailable. Download file to view."
- "Download PDF" button available
- "Retry" button to attempt reload
- Error logged for debugging

### AC7: PDF Download Option
**Given** I want to download the PDF  
**When** I click the download button  
**Then**:
- PDF file downloads to browser's default location
- Filename matches original document name
- Download doesn't interrupt current view
- Success toast: "PDF downloaded successfully"

### AC8: Responsive PDF Display
**Given** I view the PDF on different devices  
**When** the viewport changes  
**Then**:
- Desktop (≥1024px): Side-by-side with entities (40% width)
- Tablet (768px-1023px): Separate tab, full width
- Mobile (<768px): Collapsible accordion, full width
- PDF scales appropriately for screen size
- Controls remain accessible on all devices

### AC9: PDF Performance
**Given** I load a PDF document  
**When** the PDF renders  
**Then**:
- First page renders within 2 seconds
- Subsequent pages load on-demand (lazy loading)
- Smooth scrolling between pages
- No memory leaks during extended viewing
- Handles PDFs up to 10MB without crashes

### AC10: Accessibility
**Given** I use assistive technologies  
**When** I interact with the PDF viewer  
**Then**:
- All controls are keyboard accessible
- ARIA labels on all buttons
- Screen reader announces page changes
- Focus indicators visible on all controls
- Text content extractable for screen readers

## Technical Details

### Installation
```bash
npm install react-pdf@^7.7.0 pdfjs-dist@^4.0.0
```

### PDF Viewer Component
```typescript
// src/components/results/PDFViewer.tsx
import { useState } from 'react'
import { Document, Page, pdfjs } from 'react-pdf'
import { ChevronLeft, ChevronRight, ZoomIn, ZoomOut, Maximize, Download } from 'lucide-react'
import { Button } from '@/components/ui/button'
import { Skeleton } from '@/components/ui/skeleton'
import 'react-pdf/dist/Page/AnnotationLayer.css'
import 'react-pdf/dist/Page/TextLayer.css'

// Configure PDF.js worker
pdfjs.GlobalWorkerOptions.workerSrc = `//cdnjs.cloudflare.com/ajax/libs/pdf.js/${pdfjs.version}/pdf.worker.min.js`

interface PDFViewerProps {
  pdfUrl: string
  filename?: string
}

export function PDFViewer({ pdfUrl, filename }: PDFViewerProps) {
  const [numPages, setNumPages] = useState<number>(0)
  const [pageNumber, setPageNumber] = useState<number>(1)
  const [scale, setScale] = useState<number>(1.0)
  const [isFullscreen, setIsFullscreen] = useState(false)
  const [error, setError] = useState<string | null>(null)

  const onDocumentLoadSuccess = ({ numPages }: { numPages: number }) => {
    setNumPages(numPages)
    setError(null)
  }

  const onDocumentLoadError = (error: Error) => {
    console.error('PDF load error:', error)
    setError('Failed to load PDF')
  }

  const goToPrevPage = () => {
    setPageNumber((prev) => Math.max(prev - 1, 1))
  }

  const goToNextPage = () => {
    setPageNumber((prev) => Math.min(prev + 1, numPages))
  }

  const zoomIn = () => {
    setScale((prev) => Math.min(prev + 0.25, 2.0))
  }

  const zoomOut = () => {
    setScale((prev) => Math.max(prev - 0.25, 0.5))
  }

  const fitToWidth = () => {
    setScale(1.0)
  }

  const fitToPage = () => {
    setScale(0.8)
  }

  const handleDownload = () => {
    const link = document.createElement('a')
    link.href = pdfUrl
    link.download = filename || 'document.pdf'
    link.click()
  }

  const toggleFullscreen = () => {
    setIsFullscreen(!isFullscreen)
  }

  // Keyboard shortcuts
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'ArrowLeft') goToPrevPage()
      if (e.key === 'ArrowRight') goToNextPage()
      if (e.key === 'Escape' && isFullscreen) setIsFullscreen(false)
      if ((e.ctrlKey || e.metaKey) && e.key === '+') {
        e.preventDefault()
        zoomIn()
      }
      if ((e.ctrlKey || e.metaKey) && e.key === '-') {
        e.preventDefault()
        zoomOut()
      }
    }

    window.addEventListener('keydown', handleKeyDown)
    return () => window.removeEventListener('keydown', handleKeyDown)
  }, [pageNumber, numPages, isFullscreen])

  if (error) {
    return (
      <div className="flex flex-col items-center justify-center h-96 bg-gray-50 rounded-lg border border-gray-200">
        <p className="text-error mb-4">{error}</p>
        <p className="text-sm text-gray-600 mb-4">PDF preview unavailable. Download file to view.</p>
        <div className="flex gap-2">
          <Button variant="secondary" onClick={() => setError(null)}>
            Retry
          </Button>
          <Button onClick={handleDownload}>
            <Download className="h-4 w-4 mr-2" />
            Download PDF
          </Button>
        </div>
      </div>
    )
  }

  return (
    <div className={`bg-white rounded-lg border border-gray-200 ${isFullscreen ? 'fixed inset-0 z-50' : ''}`}>
      {/* Controls */}
      <div className="flex items-center justify-between p-4 border-b border-gray-200">
        {/* Navigation */}
        <div className="flex items-center gap-2">
          <Button
            variant="secondary"
            size="sm"
            onClick={goToPrevPage}
            disabled={pageNumber <= 1}
            aria-label="Previous page"
          >
            <ChevronLeft className="h-4 w-4" />
          </Button>
          <span className="text-sm text-gray-600">
            Page {pageNumber} of {numPages}
          </span>
          <Button
            variant="secondary"
            size="sm"
            onClick={goToNextPage}
            disabled={pageNumber >= numPages}
            aria-label="Next page"
          >
            <ChevronRight className="h-4 w-4" />
          </Button>
        </div>

        {/* Zoom Controls */}
        <div className="flex items-center gap-2">
          <Button variant="secondary" size="sm" onClick={zoomOut} aria-label="Zoom out">
            <ZoomOut className="h-4 w-4" />
          </Button>
          <span className="text-sm text-gray-600 min-w-[60px] text-center">
            {Math.round(scale * 100)}%
          </span>
          <Button variant="secondary" size="sm" onClick={zoomIn} aria-label="Zoom in">
            <ZoomIn className="h-4 w-4" />
          </Button>
          <Button variant="secondary" size="sm" onClick={fitToWidth}>
            Fit Width
          </Button>
          <Button variant="secondary" size="sm" onClick={fitToPage}>
            Fit Page
          </Button>
          <Button variant="secondary" size="sm" onClick={toggleFullscreen} aria-label="Fullscreen">
            <Maximize className="h-4 w-4" />
          </Button>
          <Button variant="secondary" size="sm" onClick={handleDownload} aria-label="Download PDF">
            <Download className="h-4 w-4" />
          </Button>
        </div>
      </div>

      {/* PDF Document */}
      <div className="overflow-auto p-4 bg-gray-100" style={{ maxHeight: isFullscreen ? 'calc(100vh - 80px)' : '600px' }}>
        <Document
          file={pdfUrl}
          onLoadSuccess={onDocumentLoadSuccess}
          onLoadError={onDocumentLoadError}
          loading={<Skeleton className="w-full h-96" />}
        >
          <Page
            pageNumber={pageNumber}
            scale={scale}
            renderTextLayer={true}
            renderAnnotationLayer={true}
          />
        </Document>
      </div>
    </div>
  )
}
```
