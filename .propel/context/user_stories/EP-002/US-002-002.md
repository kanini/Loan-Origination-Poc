# User Story: Fixed Top Navigation Bar Implementation

**Epic:** EP-002 - Navigation & Information Architecture  
**Story ID:** US-002-002  
**Priority:** P0 - Critical  
**Estimate:** 5 Story Points (2 days)

---

## User Story

**As a** Financial Analyst  
**I want** a fixed top navigation bar with application logo, main menu items, and user profile icon  
**So that** I can easily navigate to any major section and access my profile settings from any page

---

## Acceptance Criteria

### AC1: Navigation Bar Structure
**Given** I am on any page of the application  
**When** I view the top of the screen  
**Then**:
- Fixed navigation bar is visible at the top (z-index ensures it's above other content)
- Navigation bar has white background with subtle bottom border shadow
- Navigation bar height is 64px on desktop, 56px on mobile
- Navigation bar spans full viewport width
- Content below navigation has appropriate top padding (64px/56px)

### AC2: Logo and Branding
**Given** the navigation bar is rendered  
**When** I view the left section  
**Then**:
- Application logo "GenAI PDF Extractor" is displayed (text-based, Primary Blue #3B82F6)
- Logo uses H4 typography (20px/600)
- Logo is clickable and navigates to Dashboard (/)
- Logo has hover effect (slight opacity change)

### AC3: Main Menu Items
**Given** the navigation bar is rendered  
**When** I view the center section (desktop ≥1024px)  
**Then**:
- Three main menu items are displayed: "Dashboard", "Upload", "History"
- Menu items use Body typography (16px/400)
- Menu items are horizontally aligned with 32px spacing
- Each menu item is clickable and navigates to respective page
- Menu items have hover effect (Primary Blue color, underline)

### AC4: Active Page Highlighting
**Given** I am on a specific page  
**When** I view the navigation bar  
**Then**:
- Current page menu item is highlighted with Primary Blue color
- Active menu item has bottom border (2px solid Primary Blue)
- Non-active menu items have default gray color
- Active state persists during page navigation

### AC5: User Profile Icon
**Given** the navigation bar is rendered  
**When** I view the right section  
**Then**:
- User profile icon is displayed (circular avatar or user icon)
- Icon size is 40px × 40px
- Icon has hover effect (shadow lift)
- Icon is clickable (placeholder for Phase 2 profile menu)
- Icon has proper ARIA label "User profile"

### AC6: Responsive Mobile Navigation
**Given** I am on a mobile device (<768px)  
**When** I view the navigation bar  
**Then**:
- Hamburger menu icon (☰) is displayed on the left
- Logo is centered or left-aligned
- Main menu items are hidden
- User profile icon is displayed on the right
- Hamburger icon is 24px × 24px with proper touch target (44px × 44px)

## Technical Details

### Component Structure
```typescript
// src/components/navigation/Header.tsx
import { Link, useLocation } from 'react-router-dom'
import { Menu, User } from 'lucide-react'
import { Button } from '@/components/ui/button'
import { cn } from '@/lib/utils'

export function Header() {
  const location = useLocation()
  const [mobileMenuOpen, setMobileMenuOpen] = useState(false)

  const navItems = [
    { label: 'Dashboard', path: '/' },
    { label: 'Upload', path: '/upload' },
    { label: 'History', path: '/history' },
  ]

  const isActive = (path: string) => location.pathname === path

  return (
    <header className="fixed top-0 left-0 right-0 z-50 bg-white border-b border-gray-200 shadow-sm">
      <div className="max-w-screen-2xl mx-auto px-4 md:px-6 lg:px-8">
        <div className="flex items-center justify-between h-14 md:h-16">
          {/* Mobile hamburger */}
          <button
            className="lg:hidden p-2 -ml-2"
            onClick={() => setMobileMenuOpen(!mobileMenuOpen)}
            aria-label="Toggle menu"
          >
            <Menu className="h-6 w-6" />
          </button>

          {/* Logo */}
          <Link 
            to="/" 
            className="text-h4 font-semibold text-primary hover:opacity-80 transition-opacity"
          >
            GenAI PDF Extractor
          </Link>

          {/* Desktop navigation */}
          <nav className="hidden lg:flex items-center space-x-8">
            {navItems.map((item) => (
              <Link
                key={item.path}
                to={item.path}
                className={cn(
                  "text-base font-normal transition-colors relative pb-1",
                  isActive(item.path)
                    ? "text-primary border-b-2 border-primary"
                    : "text-gray-600 hover:text-primary hover:underline"
                )}
              >
                {item.label}
              </Link>
            ))}
          </nav>

          {/* User profile icon */}
          <button
            className="p-2 rounded-full hover:shadow-md transition-shadow"
            aria-label="User profile"
          >
            <User className="h-6 w-6 text-gray-600" />
          </button>
        </div>
      </div>

      {/* Mobile menu */}
      {mobileMenuOpen && (
        <div className="lg:hidden border-t border-gray-200 bg-white">
          <nav className="px-4 py-4 space-y-2">
            {navItems.map((item) => (
              <Link
                key={item.path}
                to={item.path}
                className={cn(
                  "block px-4 py-2 rounded-md transition-colors",
                  isActive(item.path)
                    ? "bg-primary text-white"
                    : "text-gray-600 hover:bg-gray-100"
                )}
                onClick={() => setMobileMenuOpen(false)}
              >
                {item.label}
              </Link>
            ))}
          </nav>
        </div>
      )}
    </header>
  )
}
```
