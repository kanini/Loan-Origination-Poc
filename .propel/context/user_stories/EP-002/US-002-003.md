# User Story: Breadcrumb Navigation Component

**Epic:** EP-002 - Navigation & Information Architecture  
**Story ID:** US-002-003  
**Priority:** P1 - High  
**Estimate:** 3 Story Points (1 day)

---

## User Story

**As a** Financial Analyst  
**I want** breadcrumb navigation showing my current page hierarchy  
**So that** I understand my location in the application and can navigate back to parent pages easily

---

## Acceptance Criteria

### AC1: Breadcrumb Display
**Given** I am on any page of the application  
**When** I view below the top navigation bar  
**Then**:
- Breadcrumb navigation is displayed with page hierarchy
- Breadcrumbs use Small typography (14px/400)
- Breadcrumbs are separated by chevron icon (›)
- Breadcrumbs have light gray background (#F9FAFB)
- Breadcrumb container has 8px vertical padding

### AC2: Breadcrumb Structure for Each Page
**Given** I navigate to different pages  
**When** I view the breadcrumbs  
**Then** the following hierarchies are displayed:
- Dashboard: "Home"
- Upload: "Home › Upload Document"
- Processing: "Home › Upload Document › Processing"
- Results: "Home › Results › [Document Name]"
- History: "Home › Document History"

### AC3: Breadcrumb Interactivity
**Given** breadcrumbs are displayed  
**When** I interact with breadcrumb items  
**Then**:
- All breadcrumb items except the last one are clickable links
- Last breadcrumb item (current page) is not clickable and has darker color
- Clickable breadcrumbs have hover effect (Primary Blue color, underline)
- Clicking a breadcrumb navigates to that page
- Breadcrumbs are keyboard accessible (Tab, Enter)

### AC4: Dynamic Breadcrumbs
**Given** I am on a document-specific page (Results, Processing)  
**When** the breadcrumb displays document name  
**Then**:
- Document name is truncated if longer than 30 characters
- Full document name shows on hover (tooltip)
- Document name is fetched from route parameters or API

### AC5: Responsive Breadcrumbs
**Given** I am on a mobile device (<768px)  
**When** I view the breadcrumbs  
**Then**:
- Breadcrumbs are displayed in a single line with horizontal scroll if needed
- Breadcrumb text is truncated on very small screens
- Separator icons remain visible
- Touch targets are at least 44×44px

### AC6: Accessibility
**Given** breadcrumbs are rendered  
**When** I test with assistive technologies  
**Then**:
- Breadcrumbs use semantic `<nav>` with `aria-label="Breadcrumb"`
- Breadcrumb list uses `<ol>` for ordered list
- Current page has `aria-current="page"` attribute
- Screen readers announce breadcrumb navigation correctly

## Technical Details

### Component Implementation
```typescript
// src/components/navigation/Breadcrumb.tsx
import { Link, useLocation, useParams } from 'react-router-dom'
import { ChevronRight, Home } from 'lucide-react'
import { cn } from '@/lib/utils'

interface BreadcrumbItem {
  label: string
  path?: string
  isCurrent?: boolean
}

export function Breadcrumb() {
  const location = useLocation()
  const params = useParams()

  const getBreadcrumbs = (): BreadcrumbItem[] => {
    const path = location.pathname
    
    // Dashboard
    if (path === '/') {
      return [{ label: 'Home', isCurrent: true }]
    }
    
    // Upload
    if (path === '/upload') {
      return [
        { label: 'Home', path: '/' },
        { label: 'Upload Document', isCurrent: true },
      ]
    }
    
    // Processing
    if (path.startsWith('/processing')) {
      return [
        { label: 'Home', path: '/' },
        { label: 'Upload Document', path: '/upload' },
        { label: 'Processing', isCurrent: true },
      ]
    }
    
    // Results
    if (path.startsWith('/results')) {
      const documentName = params.documentId || 'Document'
      return [
        { label: 'Home', path: '/' },
        { label: 'Results', path: '/history' },
        { label: truncate(documentName, 30), isCurrent: true },
      ]
    }
    
    // History
    if (path === '/history') {
      return [
        { label: 'Home', path: '/' },
        { label: 'Document History', isCurrent: true },
      ]
    }
    
    return [{ label: 'Home', path: '/', isCurrent: true }]
  }

  const breadcrumbs = getBreadcrumbs()

  return (
    <nav aria-label="Breadcrumb" className="bg-gray-50 border-b border-gray-200">
      <div className="max-w-screen-2xl mx-auto px-4 md:px-6 lg:px-8 py-2">
        <ol className="flex items-center space-x-2 text-sm">
          {breadcrumbs.map((item, index) => (
            <li key={index} className="flex items-center">
              {index > 0 && (
                <ChevronRight className="h-4 w-4 text-gray-400 mx-2" />
              )}
              
              {item.isCurrent ? (
                <span 
                  className="text-gray-900 font-medium"
                  aria-current="page"
                >
                  {item.label}
                </span>
              ) : (
                <Link
                  to={item.path!}
                  className="text-gray-600 hover:text-primary hover:underline transition-colors"
                >
                  {item.label}
                </Link>
              )}
            </li>
          ))}
        </ol>
      </div>
    </nav>
  )
}

function truncate(str: string, maxLength: number): string {
  if (str.length <= maxLength) return str
  return str.slice(0, maxLength) + '...'
}
```
