## Description

Implement the card view for displaying extracted entities organized by categories with support for nested objects, confidence scores, and inline editing.

---

  - Tax: Taxpayer Information, Filing Information, Income Details, Deductions, Tax Calculations

- [ ] Nested sub-sections with numbered labels (e.g., "Taxpayer 1", "Taxpayer 2")

## Technical Implementation

```tsx
export const EntityCard = ({ category, entities }) => (
  <Card className="mb-4">
    <CardHeader>
      <h3 className="text-lg font-semibold">{category}</h3>
    </CardHeader>
    <CardContent>
      {entities.map((entity, index) => (
        <div key={index} className="mb-4">
          {Array.isArray(entity) ? (
            <NestedSection items={entity} index={index + 1} />
          ) : (
            <EntityField 
              label={entity.label} 
              value={entity.value}
              confidence={entity.confidence}
              onEdit={handleEdit}
            />
          )}
        </div>
      ))}
    </CardContent>
  </Card>
);

const EntityField = ({ label, value, confidence, onEdit }) => (
  <div className="flex justify-between py-2 border-b">
    <span className="text-gray-600">{label}</span>
    <span className={value ? '' : 'text-gray-400'}>
      {value || '--'}
      {confidence && <Badge>{confidence}%</Badge>}
    </span>
  </div>
);
```
