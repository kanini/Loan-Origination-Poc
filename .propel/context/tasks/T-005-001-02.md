## Description

Integrate the chatbot with the `/api/chat/` endpoint, manage conversation history, and implement typing indicator with Markdown response rendering.

---

## Acceptance Criteria

- [ ] POST to `/api/chat/` with message, history, and extraction context
- [ ] Conversation history (rolling window of last 10 messages)
- [ ] Typing indicator (animated dots) during API call
- [ ] Markdown rendering using ReactMarkdown
- [ ] Dual-document context included when both documents available
- [ ] Error handling with retry option

---

## Technical Implementation

```tsx
import ReactMarkdown from 'react-markdown';

const useChatMutation = () => {
  return useMutation({
    mutationFn: async ({ message, history, context }) => {
      const response = await fetch('/api/chat/', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message, history: history.slice(-10), context }),
      });
      if (!response.ok) throw new Error('Chat request failed');
      return response.json();
    },
  });
};

const MessageBubble = ({ message }) => (
  <div className={`mb-4 ${message.role === 'user' ? 'text-right' : ''}`}>
    <div className={`inline-block p-3 rounded-lg max-w-[80%] ${
      message.role === 'user' ? 'bg-primary-500 text-white' : 'bg-gray-100'
    }`}>
      {message.role === 'assistant' ? (
        <ReactMarkdown>{message.content}</ReactMarkdown>
      ) : (
        message.content
      )}
    </div>
  </div>
);

const TypingIndicator = () => (
  <div className="flex space-x-1 p-3 bg-gray-100 rounded-lg w-fit">
    <span className="w-2 h-2 bg-gray-400 rounded-full animate-bounce" />
    <span className="w-2 h-2 bg-gray-400 rounded-full animate-bounce delay-100" />
    <span className="w-2 h-2 bg-gray-400 rounded-full animate-bounce delay-200" />
  </div>
);
```
