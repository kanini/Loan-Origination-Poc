## Task Overview
Configure Django CORS headers for cross-origin requests from the React frontend and set up CSRF token handling for secure API requests.

## Dependent Tasks
- None (Backend configuration task)

## Impacted Components
- `genai_pdf_extractor/settings.py` - Django settings
- `requirements.txt` - Python dependencies

## Implementation Plan
1. Install django-cors-headers package
2. Add corsheaders to INSTALLED_APPS
3. Add CorsMiddleware to MIDDLEWARE (before CommonMiddleware)
4. Configure CORS_ALLOWED_ORIGINS for development
5. Configure CSRF_TRUSTED_ORIGINS
6. Set CSRF_COOKIE_SAMESITE and CSRF_COOKIE_HTTPONLY
7. Install and configure WhiteNoise
8. Test API requests from React frontend
