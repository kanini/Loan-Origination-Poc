## Description

Create the History page with document listing, search functionality, and filter dropdowns for date range, status, and model.

---

## Acceptance Criteria

- [ ] History page displays all documents
- [ ] Search input with debounced search (300ms)
- [ ] Filter dropdowns:
  - Date Range (Today, Last 7 days, Last 30 days, All time)
  - Status (All, Complete, Processing, Error)
  - Model (All, OpenAI, Gemini)
- [ ] Sort options: Upload date, Filename, Status (asc/desc)
- [ ] Filters persist in URL query params

## Technical Implementation

```tsx
export const History = () => {
  const [search, setSearch] = useState('');
  const [filters, setFilters] = useState({ dateRange: 'all', status: 'all', model: 'all' });
  const [sort, setSort] = useState({ field: 'uploadDate', direction: 'desc' });
  
  const debouncedSearch = useDebounce(search, 300);
  
  const { data: documents } = useQuery(
    ['documents', debouncedSearch, filters, sort],
    () => fetchDocuments({ search: debouncedSearch, ...filters, ...sort })
  );

  return (
    <div className="container mx-auto px-4 py-6">
      <div className="flex flex-wrap gap-4 mb-6">
        <Input
          placeholder="Search documents..."
          value={search}
          onChange={e => setSearch(e.target.value)}
        />
        <Select value={filters.dateRange} onValueChange={v => setFilters(f => ({...f, dateRange: v}))}>
          {/* Date range options */}
        </Select>
        {/* More filters */}
      </div>
      <DocumentList documents={documents} />
    </div>
  );
};
```
