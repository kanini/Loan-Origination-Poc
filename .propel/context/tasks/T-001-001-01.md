## Task Overview
Initialize a new React 18+ project using Vite with TypeScript configuration. Set up the standard folder structure and configure the development environment with HMR and API proxy for Django backend.

## Dependent Tasks
- None (First task in the project)

- `frontend/src/` - Source code directory
- `frontend/vite.config.ts` - Vite configuration
- `frontend/tsconfig.json` - TypeScript configuration
- `frontend/package.json` - Project dependencies

## Implementation Plan
1. Create frontend directory in project root
2. Initialize Vite project with React TypeScript template
3. Install base dependencies (react, react-dom, typescript)
4. Create standard folder structure:
   - `src/components/` - Reusable UI components
   - `src/pages/` - Page components
   - `src/hooks/` - Custom React hooks
   - `src/utils/` - Utility functions
   - `src/types/` - TypeScript type definitions
5. Configure vite.config.ts with API proxy for Django backend
6. Configure TypeScript with strict mode
7. Verify HMR functionality
8. Test production build
