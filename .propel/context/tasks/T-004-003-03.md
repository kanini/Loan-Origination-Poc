# Task: Add Delete Confirmation and Empty States

**Task ID:** T-004-003-03  
**User Story:** US-004-003 - Document History & Library Management  
**Epic:** EP-004 - Results Display & Entity Management  
**Type:** Frontend  
**Estimate:** 4 hours  
**Priority:** P1 - High

---

## Description

Implement delete functionality with confirmation dialog and empty states for the History page when no documents exist or filters return no results.

---

## Acceptance Criteria

- [ ] Delete button on each document card/row
- [ ] Confirmation dialog before delete:
  - "Are you sure you want to delete this document?"
  - Document name shown
  - Cancel and Delete buttons
- [ ] Soft delete maintains audit history
- [ ] Empty states:
  - No documents: "No documents yet. Upload your first document!"
  - No search results: "No documents match your search. Try different filters."
- [ ] "Clear Filters" button resets all filters

---

## Technical Implementation

```tsx
export const DeleteDialog = ({ document, open, onClose, onConfirm }) => (
  <Dialog open={open} onOpenChange={onClose}>
    <DialogContent>
      <DialogHeader>
        <DialogTitle>Delete Document</DialogTitle>
      </DialogHeader>
      <p>Are you sure you want to delete "{document?.filename}"?</p>
      <p className="text-sm text-gray-500">This action cannot be undone.</p>
      <DialogFooter>
        <Button variant="secondary" onClick={onClose}>Cancel</Button>
        <Button variant="danger" onClick={onConfirm}>Delete</Button>
      </DialogFooter>
    </DialogContent>
  </Dialog>
);

export const EmptyState = ({ type }) => (
  <div className="text-center py-12">
    {type === 'no-documents' ? (
      <>
        <DocumentIcon className="w-12 h-12 mx-auto text-gray-400" />
        <h3 className="text-lg font-medium mt-4">No documents yet</h3>
        <p className="text-gray-500">Upload your first document to get started</p>
        <Button className="mt-4" asChild>
          <Link to="/upload">Upload Document</Link>
        </Button>
      </>
    ) : (
      <>
        <SearchIcon className="w-12 h-12 mx-auto text-gray-400" />
        <h3 className="text-lg font-medium mt-4">No results found</h3>
        <p className="text-gray-500">Try different search terms or filters</p>
        <Button variant="secondary" className="mt-4" onClick={clearFilters}>
          Clear Filters
        </Button>
      </>
    )}
  </div>
);
```
