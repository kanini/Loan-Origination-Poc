- Acceptance Criteria:
    - PDF files only accepted
    - Maximum 10MB per file
    - Duplicate filename detection
    - Inline error messages for each validation
    - Error messages clear when issue resolved
    - Shake animation on validation error
- Edge Case:
    - Files with same name but different content
    - Corrupted PDF files
