# Task: Implement Split Panel and Responsive Layouts

**Task ID:** T-004-002-02  
**User Story:** US-004-002 - PDF Viewer & Split Panel Layout  
**Epic:** EP-004 - Results Display & Entity Management  
**Type:** Frontend  
**Estimate:** 6 hours  
**Priority:** P1 - High

---

## Description

Implement the split panel layout for desktop showing entities and PDF side-by-side, with tabbed interface for tablet/mobile devices.

---

## Acceptance Criteria

- [ ] Desktop (≥1024px): Split panel
  - 60% width for entities (left)
  - 40% width for PDF viewer (right)
- [ ] Tablet/Mobile (<1024px): Tabbed interface
  - Entities tab
  - PDF Viewer tab
- [ ] Responsive table with horizontal scrolling on mobile
- [ ] Resizable panel divider (optional)

---

## Technical Implementation

```tsx
export const ResultsLayout = ({ children, pdfViewer }) => {
  const isDesktop = useMediaQuery('(min-width: 1024px)');

  if (isDesktop) {
    return (
      <div className="flex h-[calc(100vh-200px)]">
        <div className="w-3/5 overflow-auto pr-4">{children}</div>
        <div className="w-2/5 border-l">{pdfViewer}</div>
      </div>
    );
  }

  return (
    <Tabs defaultValue="entities">
      <TabsList>
        <TabsTrigger value="entities">Entities</TabsTrigger>
        <TabsTrigger value="pdf">PDF Document</TabsTrigger>
      </TabsList>
      <TabsContent value="entities">{children}</TabsContent>
      <TabsContent value="pdf">{pdfViewer}</TabsContent>
    </Tabs>
  );
};
```
