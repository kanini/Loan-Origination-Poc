    - Options: OpenAI GPT, Gemini
    - Model descriptions shown
    - Visual selection state
    - "Extract Entities" button disabled until file + model selected

## Task Overview
Create the LLM model selection interface with radio buttons or cards for OpenAI GPT and Gemini options, including descriptions and proper form state management.

## Dependent Tasks
- T-003-001-01 (Implement Drag-and-Drop Upload Component)
- T-001-002-03 (Implement Reusable Button, Card, Input Components)

## Impacted Components
- `frontend/src/components/ModelSelection.tsx` - Model selection component
- `frontend/src/pages/Upload.tsx` - Form integration

## Implementation Plan
1. Define model options array with id, name, description
2. Create ModelSelection component
3. Implement card-style selection UI
4. Add visual selection state (border highlight)
5. Integrate with form state (React Hook Form or useState)
6. Implement button disabled state logic
7. Set up file mapping for dual-document processing
8. Test model selection with form submission
