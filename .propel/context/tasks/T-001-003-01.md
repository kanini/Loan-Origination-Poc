Integrate Framer Motion for animations and implement page transitions, button hover effects, and card interactions throughout the application.

## Dependent Tasks
- T-001-001-01 (Initialize Vite React TypeScript Project)
- T-001-002-03 (Implement Reusable Button, Card, Input Components)

## Impacted Components
- `frontend/src/components/PageTransition.tsx` - Page transition wrapper
- `frontend/src/components/ui/button.tsx` - Hover effects
- `frontend/src/components/ui/card.tsx` - Hover effects
- `frontend/src/App.tsx` - AnimatePresence wrapper

## Implementation Plan
1. Install Framer Motion package
2. Create PageTransition component with fade-in animation
3. Wrap routes with AnimatePresence in App.tsx
4. Add hover effects to Button component (scale 1.02)
5. Add hover effects to Card component (shadow lift)
6. Implement prefers-reduced-motion media query hook
7. Disable animations when reduced motion preferred
8. Test transitions are within 300ms
