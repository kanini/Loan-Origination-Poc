Configure React Query for server state management and create custom hooks for all API endpoints used by the application.

- `frontend/src/App.tsx` - QueryClientProvider

## Implementation Plan
1. Install TanStack Query and DevTools
2. Create queryClient with default options
3. Wrap App with QueryClientProvider
4. Create useDocuments hook for listing documents
5. Create useRecentDocuments hook for dashboard
6. Create useDocument hook for single document
7. Create useExtractMutation for file upload
8. Create useDeleteDocument mutation
9. Create useChatMutation for chatbot
10. Add DevTools in development mode
