## Task Overview
Implement optimistic updates for edit and delete operations using React Query mutations to provide instant UI feedback.

## Dependent Tasks
- T-002-003-01 (Configure React Query with API Hooks)
- T-001-003-02 (Implement Toast Notification System)

## Impacted Components
- `frontend/src/hooks/useDocuments.ts` - Delete mutation with optimistic update
- `frontend/src/hooks/useEditEntity.ts` - Edit mutation with optimistic update

## Implementation Plan
1. Implement onMutate callback for optimistic delete
2. Cancel outgoing queries before mutation
3. Snapshot previous data for rollback
4. Update query cache optimistically
5. Implement onError to restore previous state
6. Show error toast on failure
7. Implement onSettled to invalidate queries
8. Repeat pattern for edit mutations
9. Handle concurrent mutation conflicts
