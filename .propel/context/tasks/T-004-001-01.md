# Task: Create Results Page Layout with Header and Tabs

**Task ID:** T-004-001-01  
**User Story:** US-004-001 - Results Display with Card/Table Views  
**Epic:** EP-004 - Results Display & Entity Management  
**Type:** Frontend  
**Estimate:** 6 hours  
**Priority:** P0 - Critical

---

## Description

Create the Results page layout with document information header, action buttons, and dual-document tab navigation for switching between Loan Application and Tax Return Form results.

---

## Acceptance Criteria

- [ ] Document header with: filename, upload date, model used, processing time
- [ ] Action buttons: Edit, Export, Delete
- [ ] Dual-document tab navigation:
  - "Loan Application" tab
  - "Tax Return Form" tab
- [ ] Tab state persists during session
- [ ] Tabs only show when both documents processed

---

## Technical Implementation

```tsx
export const Results = () => {
  const { id } = useParams();
  const { data: document } = useDocument(id);
  const [activeTab, setActiveTab] = useState<'loan' | 'tax'>('loan');

  return (
    <div className="container mx-auto px-4 py-6">
      <DocumentHeader document={document} />
      
      {document?.hasDualDocuments && (
        <Tabs value={activeTab} onValueChange={setActiveTab}>
          <TabsList>
            <TabsTrigger value="loan">Loan Application</TabsTrigger>
            <TabsTrigger value="tax">Tax Return Form</TabsTrigger>
          </TabsList>
        </Tabs>
      )}
      
      <EntityDisplay 
        entities={activeTab === 'loan' ? document?.loanEntities : document?.taxEntities}
      />
    </div>
  );
};
```
